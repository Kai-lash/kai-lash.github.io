Java Information Flow

◆ A web server that implements basic authentication and responds to HTTP requests using the Java Information Flow (JIF) framework.

◆ The JIF framework follows the decentralized label model. The decentralized label model requires that the security policy of each variable be stored as a label so information flow can be validated at compile-time.

◆ Languages: JIF, Java ◆ IDE: Eclipse ◆ Team size: 3
